Set image to cover its container

object-fit

img { object-fit: cover; }

background-size: cover

div { background-size: cover; }

CSS additional class

img {
    position: relative;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

.tall {
    max-width: 100%;
}

.wide {
    max-height: 100%;
}

Auto-detection
img.addEventListener('onload', function(e) {
    e.target.classList.add(e.srcElement.clientHeight > e.srcElement.clientWidth ? 'tall' : 'wide');
});
