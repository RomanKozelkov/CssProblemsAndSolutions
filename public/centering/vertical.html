Vertical centering

Main styles

    .parent {
        width: 100%;
        background: #212529;
        box-sizing: border-box;
        height: 160px;
    }

    .child {
        width: 20%;
        margin: 0 auto;
    }
    .child-content {
        padding: 20px;
        box-sizing: border-box;
        background: #f8f9fa;
    }

Equal Top and Bottom Padding

If height of parent and child elements is known.

#padding-method .parent {
    padding: 48px; /* 160px (parent) - 64px (child) / 2 */
}
Child

Using line-height

Height of parent element should be known

#line-height-method .child {
    line-height: 160px;
    vertical-align: middle;
}
#line-height-method .child-content {
    line-height: 1.5; /*reset line-height*/
    display: inline-block;
    vertical-align: middle;
}
Child

CSS Table Method

#css-table-method .parent {
    display: table;
}
#css-table-method .child {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}
#css-table-method .child-content {
    display: inline-block;
}
Child

Absolute Positioning and Negative Margin

#position-absolute-method .parent {
    position: relative;
}

#position-absolute-method .child {
    position: absolute;
    /* vertical */
    top: 50%;
    margin-top: -32px; /* -50% of child height */
    /* horizontal */
    left: 50%;
    margin-left: -60px; /* -50% of child width */
}
Child

Absolute Positioning and Translation

#position-absolute-translation-method .parent {
    position: relative;
}

#position-absolute-translation-method .child {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%,-50%);
}
Child

Absolute Positioning and Stretching

#position-absolute-stretch-method .parent {
    position: relative;
}
#position-absolute-stretch-method .child {
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    margin: auto;
    height: 64px; /* must be set */
}
Child

Flexbox

#flexbox-method .parent {
    display: flex;
    align-items: center; //vertical
    justify-content: center; //horizontal
}
Child

CSS Grid

#grid-method .parent {
    display: grid;
}
#grid-method .child {
    margin: auto;
}
Child
